# Mine vs Rock Classification using Logistic Regression

This project applies Logistic Regression to classify sonar signals as either mines or rocks.

## Methods
- Data preprocessing
- Train/test split
- Logistic Regression model
- Evaluation using classification metrics

## Result
The model achieves strong classification accuracy, demonstrating the effectiveness of logistic regression for binary classification tasks.
